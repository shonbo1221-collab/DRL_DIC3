# 任務對話紀錄

本檔案彙整了在建置此「多臂賭博機 (Multi-Armed Bandit)」專案過程中，使用者與 AI 之間的主要對話與任務歷程。

---

### 💬 第一階段：奠定專案框架與 A/B 測試
**使用者要求**：
請扮演強化學習與資料科學專家的角色，使用 Python 撰寫一段多臂賭博機（Multi-Armed Bandit）的傳統 A/B 測試模擬程式碼。
【環境設定】
總預算（步數）：10000 元。
3 台老虎機（A, B, C），其真實期望回報（Intrinsic return mean）分別為 0.8, 0.7, 0.5。
假設每次花費得到的回報服從常態分佈（標準差設為 1.0）。
【演算法邏輯：傳統 A/B 測試】
測試期（純探索）：將前 1500 元平均花費在 A、B、C 三台機器上。結算後將剩下的 8500 元全部投入「實際平均回報最高」的那一台。
【輸出要求】
使用 numpy 進行向量化運算，並使用 matplotlib 繪製雙子圖（累積平均回報、標示測試期過渡線等），具備良好的模組化與註解。

**AI 執行**：
根據此要求，建立了專案的骨幹視覺風格、常態分佈的雜訊機制，並且加入了 $\epsilon$-Greedy 與 Optimistic Initial Values 等基礎對照組與演算法的 Markdown 介紹。

### 💬 第四階段：開發進階演算法 UCB 與 Thompson Sampling
**使用者要求**：

**AI 執行**：
- **UCB**：撰寫了 `ucb_bandit.py`，模擬探索信心常數 $c$ 從極度貪婪 ($0.1$) 擴展到過度探索 ($10.0$) 的變化。同步撰寫了 `UCB_Introduction.md` 探討其「樂觀面對未知」的不確定性公式上界。
